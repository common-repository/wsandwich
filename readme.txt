== Description ==

Wsandwich Plate는 맛있는 워드프레스를 만들기 위해 꼭 필요한 플러그인입니다.
설치형 워드프레스를 사용하는 한국 사용자 분들을 위해 Wsandwich가 직접 개발한 다양한 확장 플러그인을 무료로 제공합니다.
더 맛있는 워드프레스를 위한 Wsandwich만의 레시피를 공개합니다!
(우선은 에피타이저만…)

**소셜 공유 도구**

블로그 또는 웹 사이트의 글을 소셜 미디어로 쉽게 공유할 수 있습니다.
Facebook, Twitter, Google+, LinkedIn, Pinterest, 카카오톡, 라인, 카카오스토리, 마이피플 등
공유 된 현황을 실시간 카운터를 통해 확인할 수 있습니다.
다양한 디자인 스킨을 제공합니다.
반응형 웹 기반에서도 사용이 가능합니다.

**홈페이지**

http://wsandwich.com/


== Installation ==

1. Wsandwich Plate 플러그인을 다운로드 받고 플러그인 디렉토리(/wp-content/plugins/)에 설치하세요.
2. 어드민 페이지에서 설치 된 Wsandwich Plate 플러그인을 활성화 하세요.
3. Wsandwich Plate의 확장 플러그인을 활성화하고 간단한 설정을 통해 바로 사용할 수 있습니다.
자! 지금 바로 Wsandwich와 함께 맛있는 워드프레스를 시작해보세요.


== Frequently Asked Questions ==

= 1. Wsandwich Plate가 뭔가요? =
Wsandwich Plate는 한국 워드프레스 사용자들의 더욱 편리한 사용을 위한 확장 플러그인 패키지입니다.
Facebook, Twitter, Google+ 등의 소셜 미디어로 콘텐츠를 공유하는 기능과 카카오톡, 라인, 카카오스토리 등 다양한 모바일 기반 메신저 서비스와의 연계 기능을 제공합니다.
향후, 더 맛있는 워드프레스 사용을 위한 다양한 확장 플러그인을 지속적으로 제공할 예정입니다.
자세한 사항은 Wsandwich 블로그(http://wsandwich.com/blog/)를 참고해주세요.

= 2. 설치를 했는데 제대로 작동하지 않습니다. =
Wsandwich Plate는 WordPress 3.0.1 또는 그 이상의 버전에서 정상적으로 사용이 가능합니다.
다만, 사용하시는 테마와 충돌하거나 예기치못한 오류가 있을 수 있습니다.
사용 중 문제가 발생하시면 Wsandwich team(chef@wsandwich.com)으로 문의해주세요.

= 3.  Wsandwich의 프리미엄 서비스가 있나요? =
Wsandwich는 다양한 프리미엄 서비스를 제공하고 있습니다.
자세한 사항은 Wsandwich 웹사이트(http://wsandwich.com)를 방문하시기 바랍니다.

= 4. Wsandwich Plate는 누가 만들었나요? =
Wsandwich Plate는 i4u networks, Inc.의 Wsandwich team에서 만들고 있습니다. 
‘한국 WordPress 생태계에 기여하는 집단이 되자!’라는 생각으로 불철주야 맛있는 워드프레스를 만들기 위한 레시피를 연구하고 있습니다.

= 5. Wsandwich에 참여하고 싶어요. =
Wsandwich team은 언제나 새로운 셰프를 맞이할 준비가 되어있습니다.
기획자, 디자이너, 개발자, 마케터 여러분 언제라도 연락주세요!
주방은 이미 준비되어 있습니다!
연락 : 마스터셰프(master.chef@wsandwich.com)
